# 🎓 答题生存 (Quiz Survival)

**Minecraft Fabric 模组 | 1.21.11**

> 谁说玩我的世界只能是玩游戏？

想砍树？先答题。想挖矿？先答题。想打怪？先答题。甚至跳一下，都要答题。

当我的世界和答题挂钩，一切都不一样了。题库完全开放——默认是 Minecraft 知识题，但你可以换成**高等数学、物理、英语、历史**……任何你想学的内容。边玩游戏边学习，想玩游戏？那就先答对题。

**这不是一个普通的模组，这是一个用游戏驱动学习的系统。**

---

## ✨ 特性

### 🔨 全行为拦截
- **砍伐** — 砍树触发答题
- **挖掘** — 挖矿/石头/泥土分类触发
- **战斗** — 近战和远程攻击都触发答题，答对才能造成伤害
- **跳跃** — 跳跃也要答题
- **交互** — 打开工作台、箱子、熔炉等需要答题

### ⚔️ 战斗系统
- 攻击怪物时，周围怪物全部冻结，安心答题
- 答对后根据怪物血量分级计算伤害（小怪秒杀、中怪半血、Boss固定伤害）
- 弓箭/弩等远程攻击伤害为近战的50%（可配置）
- 答错解冻怪物 + 扣血惩罚，苦力怕答错直接点燃爆炸！
- 支持PvP答题决斗——打对方玩家也触发答题

### 📊 答题机制
- **200+ 道** Minecraft 知识题，涵盖挖矿、战斗、附魔、酿造、红石、生物群系等
- 不重复出题，所有题目用完后自动重新洗牌
- 答对获得 **免答次数**（按分类独立计数），消耗完再触发
- 答错递增扣血（公式：`min(上限, 底数^连错次数)`）
- 倒计时进度条，超时自动判错

### 🎨 界面
- 答题界面带分类标签、扣血预警、倒计时进度条
- HUD左上角实时显示各分类剩余免答次数
- 答对/答错分别播放音效

### ⚙️ 全配置化
- 所有参数均可通过 JSON 配置文件调整，无需重新编译
- 支持 `/quizsurvival reload` 热重载配置和题库
- 自定义方块分类、战斗排除实体、伤害系数等
- 题库支持外部 JSON 文件，可自由添加/修改题目

### 🌐 多语言支持
- 内置中文和英文题库（各200+道题）
- 游戏内用 `/quizsurvival lang <zh|en>` 切换语言
- 所有界面文字、HUD、聊天消息、配置注释均为中英双语

### 🔒 多人安全
- 所有状态按玩家UUID隔离
- ConcurrentHashMap 保证线程安全
- 多人冻结同一怪物时引用计数，避免提前解冻

## 📦 安装

1. 安装 [Fabric Loader](https://fabricmc.net/) (1.21.11)
2. 安装 [Fabric API](https://modrinth.com/mod/fabric-api) (0.140.2+)
3. 将模组 JAR 放入 `.minecraft/mods/` 目录
4. 启动游戏，首次运行自动生成配置文件和题库

## 🎮 指令

所有指令统一使用 `/quizsurvival` 前缀：

| 命令 | 权限 | 说明 |
|------|------|------|
| `/quizsurvival` | 所有人 | 显示指令帮助 |
| `/quizsurvival reload` | OP | 热重载配置文件和题库 |
| `/quizsurvival stats [玩家]` | 所有人 | 查看答题统计（总数/正确/错误/正确率） |
| `/quizsurvival time <秒>` | OP | 修改答题倒计时，0 = 无限制 |
| `/quizsurvival lang <zh\|en>` | OP | 切换题库语言 |

## 📁 配置文件

路径：`config/quizsurvival/quizsurvival_config.json`

```json
{
  "questionLanguage": "zh",      // 题库语言: zh=中文, en=English
  "freeActionCount": 5,          // 答对获得的免答次数
  "freezeRadius": 32,            // 战斗时冻结怪物范围（格）
  "quizTimeLimit": 30,           // 倒计时秒数，0=无限
  "wrongDamageBase": 2.0,        // 答错扣血底数
  "wrongDamageMax": 20.0,        // 答错扣血上限
  "smallMobHpThreshold": 20,     // 小怪血量分界线
  "mediumMobHpThreshold": 100,   // 中怪/Boss血量分界线
  "meleeDamageSmall": 10.0,      // 小怪伤害倍率（秒杀）
  "meleeDamageMedium": 0.5,      // 中怪伤害倍率（半血）
  "meleeDamageBoss": 20.0,       // Boss固定伤害
  "rangedDamageRatio": 0.5       // 远程伤害比例
}
```

所有参数均带中英双语注释，删除配置文件重启即可恢复默认值。

## 📝 自定义题库

路径：`config/quizsurvival/quiz_questions_zh.json`（中文）或 `quiz_questions_en.json`（英文）

```json
[
  {
    "question": "钻石矿最低可以在第几层生成？",
    "options": ["-64层", "0层", "16层", "32层"],
    "answer": 0
  }
]
```

- `answer` 为正确选项的索引（从 0 开始）
- 修改后使用 `/quizsurvival reload` 热重载，无需重启
- 用 `/quizsurvival lang <zh|en>` 切换语言，缺失的题库文件会自动生成

### 💡 题库不止是 Minecraft

默认题库是 Minecraft 知识，但 JSON 格式意味着你可以放任何内容：

| 场景 | 题库内容 |
|------|----------|
| 🎓 学生复习 | 高等数学、物理公式、化学方程式 |
| 🌍 语言学习 | 英语单词、日语假名、语法选择 |
| 📚 考试备考 | 驾照科目一、公务员常识、医学基础 |
| 🏫 课堂互动 | 老师自定义题目，学生边玩边学 |
| 🎮 服务器活动 | 服务器自定义知识竞赛题库 |

只需编辑 JSON 文件，4个选项1个答案，无需编程知识。

项目已附带 **高等数学题库（50题）** 作为示例，位于 `optional_question_banks/` 目录：

```
optional_question_banks/
├── quiz_questions_zh.json   ← 中文高数题库
└── quiz_questions_en.json   ← 英文高数题库
```

使用方法：直接复制到 `config/quizsurvival/` 目录覆盖同名文件，然后 `/quizsurvival reload`。

## 🎯 玩法技巧

1. **优先答对** — 每次答对获得5次免答机会，连续答对 = 长时间自由行动
2. **别连错** — 答错扣血递增（2→4→8→16→20上限），连错很危险
3. **注意苦力怕** — 攻击苦力怕答错它会直接爆炸，建议远程攻击
4. **善用倒计时** — 不确定的题目宁可猜也不要等超时，超时也算答错
5. **弓箭伤害减半** — 远程虽安全，但伤害只有近战的50%

## ❓ 常见问题

**Q: 题目太难/太简单怎么办？**
A: 编辑 `quiz_questions_zh.json` 自定义题目，`/quizsurvival reload` 即可生效。

**Q: 怎么关闭某个行为的答题？**
A: 将对应分类的免答次数设为很大的数值（如99999），等同于关闭。

**Q: 可以在服务器上使用吗？**
A: 可以！客户端和服务端都需要安装此模组。所有玩家状态独立隔离。

**Q: 倒计时到0会怎样？**
A: 自动判定为答错，扣血并弹出下一题（战斗模式下怪物同时解冻）。

**Q: 可以和其他模组一起用吗？**
A: 兼容大部分 Fabric 模组，使用标准事件系统，无核心覆盖。
